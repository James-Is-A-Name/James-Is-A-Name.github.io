How does Javascript compare to HTML and CSS?

Javascript is a language that is intended to allow for manipulation and processing of information and input.
HTML and CSS are more static with the intent of having information and displaying it.

Can you explain control flow and loops using an example from everyday life?

One example could be doing a load of washing.
The overall flow would start with checking the machine is not in use, fill the machine, set the machines settings and starting it. Assessing an item of clothing to see if it will be put into the machine and putting it in can be seen as a the contents of a loop. Checking and moving the item in is an action that will be repeated until a set of conditions are met(e.g. no more clothes or machine full).

What is the difference between accessing data from arrays and objects?

Arrays are an index list/collection of the same thing roughly speaking. When accessing data from an array you are asking for it by location(e.g. the sixth element).
With objects you are requesting somthing that it supposidly knows. With an object you are accessing thing based on what it is called not where it is.

What are functions and why are they useful?

Functions are pieces of code that have been bundeled up into its own section that can be called in other parts of code to complete its task.
They are useful for both reusing code and also for conceptual organisation (e.g. applyIntrest(bankAccount,interestRate) shows more intent and purpose than bankAccount.balance += bankAccount.balance*interestRate).
