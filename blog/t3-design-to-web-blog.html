What is a response site and why is it important?

A responsive website is one that adapts to how it is viewed be it a small mobile phone or a large computer display.
The importance is to accommodate the fact that people interact with things in different but will still want the same information.

What is mobile first design and why is it important?

Mobile first design is having the Mobile design be the prime focus when starting.
The reason for starting with mobile is that it is small with space being a premium so it forces the design to be stripped down to essentials. The idea is its easier to add bonus parts than to remove things which might be vital.

What are frameworks?

Frameworks are tools for a web page that help by giving foundations for common elements and designs.
The pros of this is less time needed on fidly little parts to get a passable design in place. the cons are it add its own complexity that needs to be learned and can be restrictive when trying new designs

What is a wireframe and why use it?

A wireframe is a barebones example of a page with the physical layout of items in place.
It is useful as a guide as it can be quickly created and then used as a reference to aim for. It also helps identify the important aspects of a design.
