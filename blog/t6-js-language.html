To a non-tech friend explain ...

Parentheses ( )

Parentheses are what are used to tie information to a command if it needs it. If you asked someone to repeat after you the part you wanted repeated would be in parentheses repeat after me ("Some words").

Brackets [ ]

Brackets are used talking about something in an object or an array. An example could be a person, asking person["age"] would get the response of the persons age (if they knew it).

Braces { }

Braces are used for defineing an object and for defining an area of code. Defining an area of code is useful when you want to reuse code. An example would be mixing together something like icing for a cake, You could tell someone how to stir once {move arm holding a spoon from the top of the container
to the side then the bottom then back up} and then tell them to repeat those actions rather than saying them again and again.

Single quotes ' '

Single quotes are used for declearing a sentence as a thing very similar in a book or other text. For example the instruction "tell someone the message 'The cat is back'"" means that the message is 'The cat is back' and telling someone involves saying 'The cat is back' not 'the message the cat is back'

Double quotes " "

In javascript double quotes are very similar to use as single quotes. The difference between the two is more of a style/layout decision than a purely functional one. In the single quotes example the double quotes were used to encapsulate the single quotes but doing the opposite is also allowed the only difference being the inner quote is in double quotes rather than single quotes.
