What 'scope' is, and how it works in Javascript?

scope is refering to where a variable/function/thing is logically located and where it can be seen, known to exist. for instance a global variable has a global scope which means it can be seen and used anywhere in the javascript code.
This can cause issue if not carefull as poorly chosen global variable names and managment as a variable might be accedentally reused in seperate parts of the code leading to erratic behaviour. Think online shopping but the total cost of the shopping cart and the cost of items you are looking at getiing swapped.

Scope can be reduced to sections of code such as within functions. This allows for reuse of logical variable names without fear of mixing issues (count is a very useful description for whats wanted in many places).
